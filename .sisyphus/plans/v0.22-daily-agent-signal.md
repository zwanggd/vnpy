# Agent v0.22 — daily_agent_signal 后处理优化

> **Status**: 🚧 开发中 | **Plan created**: 2026-05-26

## TL;DR

在不改 AgentSignal schema、不重跑 LLM 的前提下，优化 daily_agent_signal 的计算方式：
- 单条信号: direction_sign × strength^1.2 × conf_nonlinear × relation_weight × horizon_weight
- 多模型: agreement-based ensemble (同一条新闻多个模型的 consensus 加权)
- 事件去重: 基于 event 字段 normalize → event_key → median 聚合
- 日级聚合: event_count 替代 news_count 作为 SQRT 分母，tanh 替代 clip
- Mixed 风险惩罚: mixed 信号不进入方向分，但降权日级信号
