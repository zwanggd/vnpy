# 登录使用

每次启动 VeighNa Fusion 后，系统会先进入登录界面。用户需输入账号信息，并确认已阅读相关协议，登录成功后才能进入主程序。

## 首次启动界面

启动 Fusion 后，界面通常会显示：

- 用户名输入框；
- 密码输入框；
- 登录按钮；
- 退出按钮；
- 用户协议和隐私政策的勾选项。

建议首次使用时先完整浏览一遍登录界面，确认账号、网络和协议勾选状态后再点击登录。

![](https://vnpy-doc.oss-cn-shanghai.aliyuncs.com/fusion/8.png)

## 输入账号密码

在登录界面中：

- 在用户名输入框中填写所属期货公司为你开通的用户名；
- 在密码输入框中填写对应密码；
- 确认信息无误后，再执行登录操作。

如你不确定账号信息来源或权限范围，应优先联系所属期货公司确认，不建议反复尝试未知账号或密码。

## 阅读并同意相关协议

Fusion 登录前要求用户勾选相关协议，表示已阅读并同意：

- 《用户协议》
- 《隐私政策》

若未勾选协议，系统将不会继续登录流程。

![](https://vnpy-doc.oss-cn-shanghai.aliyuncs.com/fusion/9.png)

## 首次登录修改密码

部分账号在首次登录时会要求立即修改默认密码。若出现修改密码提示，请按要求完成以下操作：

1. 输入新密码；
2. 再次输入新密码确认；
3. 提交修改；
4. 修改成功后，使用新密码继续后续登录流程。

设置新密码时，建议遵循所属期货公司的密码规范，并避免与旧密码保持一致。

![](https://vnpy-doc.oss-cn-shanghai.aliyuncs.com/fusion/10.png)

## 更新提示说明

Fusion 在登录前可能检查是否存在新版本。若检测到更新，系统通常会弹出更新提示窗口，显示当前版本与最新版本信息。

你可以根据实际情况选择：

- 立即更新后再重新启动软件；
- 暂时跳过本次更新，先继续登录。

如果所属期货公司对升级有统一安排，请优先遵循统一通知。

## 常见登录问题

### 提示未勾选协议

现象：点击登录后，系统提示需要先同意相关协议。  
处理建议：勾选协议后再重新登录。

### 提示用户名或密码错误

现象：登录失败，并提示账号信息不正确。  
处理建议：检查输入内容是否有空格、大小写、标点符号切换错误或复制错误；如仍无法登录，请联系所属期货公司确认账号状态。

### 提示首次登录需要修改密码

现象：登录校验通过后，系统要求立即修改密码。  
处理建议：按页面提示完成密码修改，修改成功后继续登录。

### 提示无法连接服务器

现象：登录时出现网络连接失败、服务器不可达或类似提示。  
处理建议：先检查本机网络状态，再确认当前网络环境是否允许访问 Fusion 所需服务。

### 提示该账户已在其他设备登录

现象：系统提示该账号正在其他设备使用，当前无法继续登录。  
处理建议：先在原设备退出，或等待原有会话失效后再重试；如长期无法恢复，请联系所属期货公司或技术支持协助处理。

### 提示授权或许可异常

现象：登录过程中提示授权无效、授权超限或类似信息。  
处理建议：通常与账号授权状态相关，应由所属期货公司或服务提供方协助核查。

## 登录成功后做什么

登录成功后，建议按以下顺序继续操作：

1. 阅读 [主界面说明](fusion_trader.md)，了解菜单和组件位置；
2. 阅读 [CTP交易连接](fusion_ctp.md)，完成账户连接和行情订阅；
3. 再进入 `CTA策略`、`回测` 和 `智策` 章节，逐步建立自己的研究与交易流程。
