# 产品介绍

VeighNa Fusion 是面向期货公司经纪业务客户的 Windows 量化交易与智能投研软件。用户可以通过单一 `CTP` 接口完成行情查看、手工交易、`CTA策略` 运行、历史回测、参数优化以及 `智策` 辅助研究等工作。

与社区版偏向开源框架、Elite版覆盖更广泛专业场景不同，Fusion 更强调安装简单、流程清晰和上手直接，适合刚开始接触量化交易或希望在图形界面中逐步建立研究流程的用户。

## VeighNa Fusion是什么

VeighNa Fusion 建立在 VeighNa 量化交易体系之上，提供开箱即用的图形化桌面终端。用户启动软件后，可先完成账号登录，再连接交易账户，随后使用内置模块开展交易与研究工作。

Fusion 的核心特点包括：

- 仅提供 Windows 版本，采用安装包方式交付；
- 仅支持单一 `CTP` 交易接口，减少入门阶段的选择成本；
- 内置 `CTA策略`、`回测`、`数据管理` 和 `风控` 等常用能力；
- 提供 `智策` 能力，支持策略思路整理、代码辅助、回测与优化流程衔接。

## 适合哪些用户

VeighNa Fusion 主要适合以下用户：

- 通过期货公司获取 Fusion 服务，希望在本地桌面软件中完成量化交易与研究工作的客户；
- 已具备基本期货交易知识，正在从手工交易逐步过渡到程序化交易的用户；
- 希望在图形界面中完成策略研究、回测和参数优化，而不是从源码环境开始搭建的用户；
- 希望借助 `智策` 提升策略研究和代码整理效率，但仍愿意自行完成审查和验证的用户。

如果你已经非常熟悉 Python 开发环境，且更关注框架扩展、接口开发或源码级定制，可以进一步参考 [VeighNa 社区版文档](https://www.vnpy.com/docs/cn/community/index.html)。

## 可以完成哪些事情

使用 VeighNa Fusion，通常可以围绕以下主线开展工作：

1. 安装并启动软件；
2. 登录 Fusion 账号；
3. 连接 `CTP` 交易账户；
4. 查询合约、订阅行情、进行手工下单；
5. 使用 `CTA策略` 模块加载和运行策略；
6. 使用回测模块验证策略表现并进行参数优化；
7. 使用 `智策` 辅助梳理策略逻辑、生成代码和分析结果；
8. 结合 `风控` 模块和人工复核，逐步完善研究与交易流程。

建议初次使用时，先按 [安装指南](fusion_install.md)、[登录使用](fusion_login.md) 和 [CTP交易连接](fusion_ctp.md) 的顺序完成基础配置，再进入策略和智策章节。

## 与社区版和Elite版的区别

从文档使用角度来看，三者的定位可以简单理解为：

| 版本 | 主要特点 | 更适合谁 |
| --- | --- | --- |
| 社区版 | 开源、可扩展、强调源码与生态 | 有开发经验、希望自行定制框架的用户 |
| Elite版 | Windows 成品软件、功能覆盖较广 | 需要成熟桌面交易与研究工具的专业用户 |
| Fusion版 | Windows 安装包、单一 CTP、入门友好、强调智策辅助 | 期货公司经纪业务客户中的入门和初阶量化用户 |

Fusion 文档因此会更关注“如何完成一条完整使用路径”，而不是罗列全部底层配置项或框架原理。

## 使用前需要了解的事项

在正式使用 VeighNa Fusion 前，建议先明确以下几点：

- Fusion 提供的是量化交易与智能投研工具，不构成投资建议、投资顾问服务或收益承诺；
- `智策` 属于辅助功能，生成的策略逻辑、代码、回测结果和优化建议都需要用户独立审查；
- 回测结果、参数优化结果和历史表现均不代表未来实盘收益；
- 未经充分验证的策略、参数和自动化流程，不应直接用于实盘交易；
- 账号开通、权限变更和部分服务范围，通常以所属期货公司的安排为准。

关于 Fusion 中 AI 功能的边界与注意事项，可继续阅读 [智策使用须知](../agent/fusion_agent_notice.md)。
