# Signal Recommendation: Agent Alpha Event Study (CATL 300750)

**Date**: 2026-05-16
**Sample**: 1,514 news events with paired DeepSeek v4 Flash + Qwen 3.6-35B analysis
**Underlying**: CATL (300750) daily bars, 2020-01-02 to 2026-05-12

---

## Conclusion: 不建议进入策略回测

经过事件研究方法验证，**10个信号均未展示出alpha预测能力**。核心问题：多头信号（positive）与空头信号（negative）的后续收益率没有方向性差异，甚至空头信号的收益率更高。

### Key Evidence: Counterintuitive Returns

| Consensus Direction | N | T+5 Mean Return | T+5 Median | T+5 Win Rate |
|---|---|---|---|---|
| **Positive** | 60 | **+2.85%** | +2.70% | 65.00% |
| **Negative** | 46 | **+3.05%** ⚠️ | +3.40% | 73.91% |
| **Neutral** | 1,404 | +0.85% | +0.84% | 55.84% |
| Overall (all events) | 1,510 | +1.00% | +0.84% | 56.75% |

⚠️ 负面共识信号的T+5平均收益率（+3.05%）高于正面信号（+2.85%），且胜率更高（73.91% vs 65.00%）。这完全违背了信号应有的预测方向。

### Why This Happens

1. **CATL强势牛市**：2020-2026年期间，CATL股价从107涨至431，年化涨幅约32%。在此期间几乎任何事件日后都倾向上涨。
2. **Agent误判新闻性质**：将H股配售、股东捐赠等公告标记为"negative"，但市场可能将其解读为利好（如配售用于扩张产能）。
3. **样本量不足**：正面共识仅60条，负面仅46条，且高度集中于少数交易日（如2020-01-02有289条新闻共享同一entry_date）。

## Caveats

1. **新闻覆盖不均衡**：多数新闻来自公告（cninfo），缺乏市场新闻、研报等多样化来源
2. **Entry_date偏差**：大量新闻共享同一entry_date（如周末/假期后首个交易日），导致样本独立性存疑
3. **牛市周期偏差**：整个回测期恰逢CATL主升浪，无法检验信号在熊市中的表现
4. **Agent模型局限**：当前使用本地小模型（Qwen 3.6B），其新闻解读能力可能存在系统性偏差
5. **仅覆盖一只股票**：CATL作为行业龙头，其走势与行业/大盘高度相关，信号可能混杂了市场beta

---

## Next Steps (Before Backtesting)

1. **扩大标的范围**：至少覆盖沪深300成分股，检验信号在横截面上的分化能力
2. **引入市场中性化**：使用残差收益率（r_i − r_market）替代原始收益率进行事件研究
3. **升级Agent模型**：使用更大参数量的模型或API模型进行新闻解读
4. **丰富新闻来源**：加入财经媒体新闻、社交媒体情绪等非公告类信息
5. **时间窗口优化**：考虑entry_date重叠问题，按事件日进行去重或加权处理
